# ƏDV Hesabat Generatoru

ƏDV (НДС) bəyannamə XML fayllarından aylıq hesabat yaradan veb tətbiq.

## İstifadə

1. Saytı açın
2. ƏDV bəyannamə XML fayllarını yükləyin (bir və ya bir neçə)
3. **Hesabat yarat** düyməsinə basın
4. Hesabatı ekranda görün, CSV-yə ixrac edin və ya çap edin

## Hesabat bölmələri

- **Hissə 1** — Hesabat dövründəki dövriyyə üzrə ƏDV hesablanması (301–305)
- **Hissə 2** — ƏDV-nin əvəzləşdirilməsi
- **Hissə 3** — Debitor borc
- **Hissə 4** — Hesablaşma (ödənilməli / qaytarılmalı)
